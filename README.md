# Bootcamp-Santander-2025---Ciência-de-Dados-com-Python-ETL

## Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio "Explorando IA Generativa em um Pipeline de ETL com Python" dentro do Bootcamp Santander 2025 - Ciência de Dados com Python.

O desenvolvimento seguiu as instruções e estruturas apresentadas nos vídeos do desafio, com algumas personalizações e adaptações necessárias:

- **Nomes dos usuários**: Customizados para os dados de teste
- **Adaptações no código**: Realizadas devido à indisponibilidade da API original
- **Tipo de mensagem**: Personalizado para a geração de conteúdo pela IA

## O que é ETL?

ETL é um processo fundamental em engenharia de dados que significa:
- **Extract (Extrair)**: Coleta de dados de diversas fontes
- **Transform (Transformar)**: Processamento e modificação dos dados para o formato desejado
- **Load (Carregar)**: Inserção dos dados transformados no destino final

O objetivo deste pipeline ETL é automatizar o processo de extração de dados de usuários, transformá-los utilizando IA Generativa para criar conteúdo personalizado, e então carregar essas informações de volta ao sistema.

## Tecnologias Utilizadas

- **Python**: Linguagem principal do projeto
- **Pandas**: Manipulação e análise de dados
- **OpenAI API**: Integração com IA Generativa para criação de conteúdo personalizado
- **Requests**: Requisições HTTP para APIs
- **JSON**: Formato de dados para armazenamento e transferência
